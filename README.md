
# Projeto consulta clima

Este projeto consulta o clima a partir do nome da cidade que o usuário digitar.

Utilizei a api open weheather para colher os dados da cidade desejada.
## Imagem do projeto:

## Aprendizados:

Aprendizados empregados no projeto: 

* Código Assíncrono: async, await, fetch
* Manipulação de elementos através do DOM
* manipulação e consumo de dados externos via api
* Manipulação de JSON
